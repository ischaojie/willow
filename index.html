Hello, Willow is a theme designed for the static site generator tool zola. Without beautiful components like Tabler, Willow would not exist. Thanks to Tabler and other third-party libraries, thank you for open source!

Some main features of Willow:

  • Common blog functions such as article lists, tag pages, about pages, RSS feeds, etc.
  • Supports the giscus commenting system.
  • Supports photography portfolio pages (likes support coming soon).
  • Supports memos feature.

For detailed usage instructions, please refer to the README. If you have any questions, feel free to raise an issue.
